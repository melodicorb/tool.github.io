Frequently Asked Questions

Is my data secure?

Yes, all files are encrypted during transfer and storage. We use secure HTTPS connections to protect your data.

How long are my files stored?

Files are automatically deleted after the expiry time you set (default is 24 hours). You can also manually delete files at any time.

What types of files can I share?

You can share most common file types including documents, images, videos, and archives. However, executable files (.exe) are not permitted for security reasons.

Is there a file size limit?

Yes, the maximum file size is 100MB per file. For larger files, consider compressing them or using a file splitting tool.
